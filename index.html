CSS Snippet Cheat Sheet

Changing the Border Corners
border-radius: 25px
Rotating Items
.rotate {
transform: rotate(7deg); }
Flexbox Element
/* Turns this element into a flexbox */
display: flex;
/* Forces elements to the next line if they don't fit */
flex-wrap: wrap;
Display text only when the mouse hovers over the parent element
.[class] [child element] {
display: none;}

.[class]:hover [child element]{
    display:block;}
Tooltip
[HTML span class = tooltip] .tooltip {
    position: relative;
    display: inline-block;
    cursor: pointer;}
    
[HTML span class = tooltip] .tooltip:hover::after{
    content: "text goes here";
    background-color: #13293d;
    color: #fff;
    padding: .25em;
    font-size: .8em;
    position: absolute;
    top: -20px;
    min-width: 200px;}
Adding Symbols (Entities) before/after text
[parent] [child]::before,
[parent] [child]::after {
content: '\26F5';
color: #8ac4ff;
padding: 1%;
font-family: cursive;}
